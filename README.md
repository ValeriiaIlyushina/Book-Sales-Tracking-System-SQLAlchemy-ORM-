# Система учёта продаж книг с использованием SQLAlchem
Проект представляет собой систему учёта издателей, книг и фактов их продаж, реализованную с использованием ORM SQLAlchemy и реляционной базы данных (PostgreSQL).

В рамках проекта были спроектированы и реализованы модели данных, отражающие связи между издателями, книгами, магазинами и продажами. Реализована логика загрузки тестовых данных из JSON-фикстур, а также механизм выборки данных по заданному издателю.

Программа позволяет по имени или идентификатору издателя получить список фактов продаж его книг с указанием названия книги, магазина, стоимости и даты покупки. Запросы к базе данных выполняются с использованием ORM-подхода без написания SQL вручную.
